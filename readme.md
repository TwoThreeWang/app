# FastApi_demo 说明

使用 FastApi 创建的示例文件，集成了用户管理功能和 TODO 示例，基于 Token 认证。

# 启动方法

1、终端执行以下命令

`uvicorn main:app --reload`

2、ide 运行 main.py 文件

3、Docker 运行

# 使用

启动成功后访问 127.0.0.1:8000
```
{
"message": "Hello World"
}
```

具体接口和说明请访问：http://127.0.0.1:8000/docs
